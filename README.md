# 🍱 EAT & GO - 持続可能なイベント会場注文・管理システム

> 🎪 お祭りやイベント会場での行列・ゴミ問題を解決する統合システム

## 🌟 概要

「EAT & GO」は、お祭りやイベント会場での屋台の行列やごみ処理の問題を解決するために開発された革新的なシステムです。来場者・出店者・運営側それぞれに利便性をもたらし、持続可能で誰もが快適に楽しむことができる祭りの実現を目指します。

### 🎯 解決する課題

- 🚶‍♂️ 屋台の長い行列による来場者の負担
- 🗑️ ごみ処理・分別の運営負担
- 💰 キャッシュレス決済による衛生的で効率的な運営
- 🌱 環境負荷軽減と会場美化の両立

## 🔄 システム動作フロー

```
① スマートフォンで注文 📱
    ↓
② 注文受付・調理・QRシール発行 🍳
    ↓
③ 商品受け取り 🛍️
    ↓
④ ゴミ箱状況確認・ポイント還元確認 📍
    ↓
⑤ QRコードでごみ分別投棄 🗂️
    ↓
⑥ 正しく分別できればでポイント付与 🎁
    ↓
⑦ 運営側でゴミ情報確認・回収 🚛
    ↓
⑧ ポイント景品交換 🏆
```

## ✨ 主な機能

### 👥 来場者向け機能

- 🍽️ **デジタルメニュー**: 商品一覧を店舗ごとに閲覧
- 🛒 **スマートカート**: 商品をカートに追加し、数量・サイズ・テイクアウト選択
- 💳 **キャッシュレス決済**: 現金・クレジット・PayPayでの衛生的な決済
- 📱 **リアルタイム通知**: 商品準備完了をリアルタイムで通知
- 🗺️ **ゴミ箱マップ**: ゴミ箱の位置と満杯度をリアルタイム表示
- 🎁 **ポイントシステム**: 正しい分別でポイント獲得・景品交換
- 📢 **イベント情報**: 最新のお知らせやイベント情報を確認

### 🏪 出店者向け機能

- 📋 **注文管理**: リアルタイムでの注文受付・ステータス管理
- 🖨️ **QRラベル印刷**: 62mmの小型用紙にプリンターでQRシール自動発行
- 📊 **売上分析**: リアルタイム売上確認・分析レポート
- 🍽️ **メニュー管理**: 商品の追加・編集・在庫管理・公開設定
- 📅 **営業管理**: 営業時間・休業日の柔軟な設定

### 🔧 運営者向け機能

- 🎛️ **統合ダッシュボード**: 全体の注文・ゴミ状況を一元管理
- 👤 **ユーザー管理**: 来場者・出店者情報の管理
- 🗑️ **廃棄物管理**: ゴミ箱の満杯度・分別状況をリアルタイム監視
- 📝 **お知らせ管理**: リッチテキストエディターでイベント情報発信
- 🏪 **マルチ会場対応**: 複数会場・複数出店者の一括管理

## 🏗️ システム構成

### 🔧 技術スタック

- **🌐 フロントエンド**: Next.js 14 (App Router) + TypeScript
- **🎨 スタイリング**: Tailwind CSS
- **🔐 バックエンド**: Supabase (認証・リアルタイムDB・ストレージ)
- **☁️ ホスティング**: Vercel (Webアプリ)
- **🤖 IoT**: Raspberry Pi (ゴミ箱監視カメラ)
- **📡 通信**: 無線通信によるリアルタイムデータ送信
- **🎯 UI/UX**: Lucide React Icons + Framer Motion
- **📧 通知**: Brevo (メール送信) + React Toastify

### 🏛️ アーキテクチャ

```
📱 スマートフォン/タブレット
        ↕️
🌐 Vercel (Next.js Webアプリ)
        ↕️
🔐 Supabase (リアルタイムDB)
        ↕️
📡 Raspberry Pi (ゴミ箱監視)
```

**三層連携アーキテクチャ**で構成され、全ユーザーがVercelにホストされたWebアプリを通じてアクセスし、注文から廃棄まで一貫した祭り運営の効率化を実現します。

## 🚀 セットアップ方法

### 📋 前提条件

- Node.js 18以上
- npm または yarn
- Supabaseアカウント
- Vercelアカウント（デプロイ用）

### 💻 ローカル開発環境

```bash
# 📥 リポジトリのクローン
git clone https://github.com/omuct/eat-and-go.git
cd eat-and-go

# 📦 依存関係のインストール
npm install

# 🔧 環境変数の設定
cp .env

# 📝 .env.localファイルを編集
NEXT_PUBLIC_SUPABASE_URL=your_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_KEY

DATABASE_URL="your URL"
DIRECT_URL="your URL"

NEXT_PUBLIC_SITE_URL=http://localhost:3000

PAYPAY_CLIENT_ID=your_ID
PAYPAY_API_KEY=your_key
PAYPAY_CLIENT_SECRET=your_secret_key
PAYPAY_SECRET=your_secret_key
PAYPAY_MERCHANT_ID=your_id
MERCHANT_ID=your_id

NEXTAUTH_URL=http://localhost:3000
NEXT_PUBLIC_BASE_URL=http://localhost:3000

BREVO_API_KEY=your_API_KEY

# 🎯 開発サーバーの起動
npm run dev
```

### 🗄️ Supabaseデータベース設定

#### 必要なテーブル:

- 🍽️ **foods**: メニュー商品情報
- 📢 **announcements**: お知らせ・イベント情報
- 🛒 **cart_items**: ユーザーのカート情報
- 📦 **orders**: 注文情報
- 📋 **order_details**: 注文詳細情報
- 👤 **profiles**: ユーザープロフィール（来場者・出店者・運営者）
- 🏪 **stores**: 出店者情報
- 📅 **business_closures**: 営業カレンダー
- 🗑️ **waste_bins**: ゴミ箱情報・状態
- 🎁 **points**: ポイント管理
- 🏆 **rewards**: 景品管理

## 🎯 主要な特徴

### 🌟 革新的機能

- ✅ **ゼロ接触注文**: 完全キャッシュレス・QRコード活用
- 🔄 **リアルタイム同期**: Supabaseによるリアルタイムデータ更新
- 🤖 **IoT連携**: Raspberry Piカメラによるゴミ箱自動監視
- 🎁 **ゲーミフィケーション**: ポイント・景品システムで分別促進
- 🖨️ **62mm印刷対応**: コンパクトなQRラベル印刷システム
- 📱 **PWA対応**: アプリのようなスマートフォン体験

### 🌱 持続可能性

- ♻️ **分別促進**: ポイントシステムによる環境意識向上
- 📊 **データ分析**: ゴミ発生量・分別率の可視化
- 🚛 **効率的回収**: リアルタイム満杯度監視による最適な回収タイミング
- 💚 **環境負荷軽減**: デジタル化による紙使用量削減

## ⚠️ 利用制限・注意事項

- ⏰ テイクアウト注文は11:30までのみ受け付け
- 🛒 カートには最大5個までしか商品を追加できません
- 📦 同じ商品は最大3個までしか注文できません
- 🖨️ QRラベル印刷は調理完了後のみ利用可能
- 📡 ゴミ箱監視にはインターネット接続が必要
